## Overview

This repository contains the complete, reproducible analysis pipeline for the manuscript:

> *"Information Entropy of Local Coordination Environments as a Geometric Predictor of Inorganic Synthesizability"*

**Central claim:** The Shannon entropy of the coordination environment (CE) distribution within a crystal structure — computed entirely from geometry, without any electronic structure calculation — is a statistically significant predictor of thermodynamic synthesizability (energy above convex hull) across the inorganic materials database, independent of chemical composition.

### Why this is novel

Synthesizability prediction is one of the most-cited open problems in computational materials science. Existing approaches require:
- Formation energy (DFT, computationally expensive)
- Chemical similarity to known materials (database-dependent)
- Machine-learning models trained on experimental databases (black-box)

This work introduces the first **pure geometric, information-theoretic descriptor** of structural complexity as a synthesizability signal. The physical argument: structures requiring many distinct coordination environments are geometrically frustrated in a way that makes crystal growth kinetically and thermodynamically unfavorable.

## Data

Crystal structures are downloaded from the [Materials Project](https://materialsproject.org) via the `mp-api` Python client. The pipeline applies stratified sampling across three synthesizability classes:

| Class | Label | Criterion | Full DB count |
|-------|-------|-----------|---------------|
| 0 | Stable | energy\_above\_hull = 0 eV/atom | ~20,900 |
| 1 | Metastable | 0 < energy\_above\_hull ≤ 0.1 eV/atom | ~33,800 |
| 2 | Unstable | energy\_above\_hull > 0.1 eV/atom | ~33,100 |

**Filters applied:** 2–6 distinct element types; 4–80 atoms per unit cell.  
**Sample size:** 1,500 per class → **4,500 structures** total (default).

---

## Descriptors

For each crystal structure, four CE-based complexity descriptors are computed:

| Descriptor | Symbol | Definition |
|------------|--------|------------|
| CE entropy | *H* | −Σ *p*ᵢ log(*p*ᵢ) over CE-type frequency distribution |
| Distinct environments | *K* | Number of unique CE symbols present |
| Dominance | *d* | Fraction of sites with the most common CE type |
| Gini coefficient | *G* | Gini inequality measure of CE frequency distribution |

CE classification uses pymatgen's `ChemEnv` module (`LocalGeometryFinder` + `SimplestChemenvStrategy`), which maps each atomic site to one of ~30 geometry types (O:6 octahedral, T:4 tetrahedral, C:12 cuboctahedral, L:2 linear, etc.).

## Statistical Analyses

1. **One-way ANOVA** — CE entropy vs. 3-class synthesizability label  
   → Effect size η², post-hoc Tukey HSD, Benjamini–Hochberg FDR correction

2. **Logistic regression** — binary synthesizability (experimental vs. computational)  
   → Three models compared: naive (size+composition), entropy-only, full  
   → ROC-AUC with bootstrap 95% CI; McFadden pseudo-R²

3. **Spearman correlation** — CE entropy vs. energy\_above\_hull (continuous)

4. **Composition control** — per-element CE entropy baseline subtracted  
   → Tests whether structural entropy predicts synthesizability *beyond* composition

5. **Crystal-system subgroup ANOVA** — relationship tested within each crystal family

6. **Pettifor chemical-scale subgroups** — structures binned by Mendeleev-number span  
   → Tests whether the effect holds across chemically similar vs. diverse compounds

## Reproducing the Analysis

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Run the full pipeline

```bash
python main.py --api-key YOUR_MP_API_KEY
```

Get a free API key at [materialsproject.org/api](https://materialsproject.org/api).

### 3. Skip expensive steps on re-runs

The pipeline caches intermediate data:

```bash
# Re-use downloaded structures, recompute ChemEnv
python main.py --api-key YOUR_KEY --skip-download

# Re-use all cached data, re-run analysis + figures only
python main.py --api-key YOUR_KEY --skip-download --skip-chemenv
```

### 4. Full CLI options

```
--api-key        MP API key (required)
--n-per-class    Structures per synthesizability class   [1500]
--workers        Parallel ChemEnv worker processes       [8]
--timeout        Per-structure timeout in seconds        [120]
--cache-dir      Directory for cached data               [data/]
--fig-dir        Figure output directory                 [figures/]
--results-dir    Results table directory                 [results/]
--seed           Random seed for reproducibility         [42]
--log-level      DEBUG / INFO / WARNING                  [INFO]
```

### 5. Run tests

```bash
python -m pytest tests/ -v
```
